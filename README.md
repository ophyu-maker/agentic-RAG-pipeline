# Agentic-RAG-pipeline
Hands-on RAG engineering practice project based on a guided course.

## Project Overview

This project is my hands-on implementation of a production-style Agentic RAG pipeline based on the Agentic RAG course using arXiv papers as the data source. 
The goal is to understand how RAG systems are built with data ingestion, storage, search indexing, and retrieval APIs.

## Technology Stack

- FastAPI
- PostgreSQL
- OpenSearch
- Apache Airflow 3.0
- Jina AI
- Ollama
- Redis
- Langfuse
